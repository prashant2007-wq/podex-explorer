About Pokédex Explorer

Pokédex Explorer is a modern browsing application featuring the original 151 Pokémon from Generation I.

Features:

  • Search Pokémon by name or ID
  • Filter by type (18 different types)
  • Sort results using multiple options
  • Save Pokémon to a favorites list
  • Toggle between dark and light themes
  • Responsive layout for all screen sizes

Data Source:

All Pokémon data is fetched from the PokéAPI, a free and open-source Pokémon API.
